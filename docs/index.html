En kolay kontrol

Ilk acilista once Mikrofon/Ses Karti Testi (5 sn) ile Peak degerini kontrol edin.

macOS Ilk Acilis

  1. Uygulamayi acin.
  2. Mikrofon izni gelirse Izin Ver deyin.
  3. Masaustu klasoru izni gelirse yine Izin Ver deyin.
  4. Mikrofon/Ses Karti Testi (5 sn) ile kisa test yapin.
  5. Peak=0.000 degilse, Quick Kayit (Preset, Sorusuz) ile dogrudan kayda gecebilirsiniz.
