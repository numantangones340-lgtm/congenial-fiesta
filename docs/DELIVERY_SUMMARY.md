# Teslim Ozeti

Bu belge, projenin su anki teslim durumunu tek yerde toplar.

## Mevcut Durum

Proje yayinlanabilir durumdadir.

Tamamlanan ana alanlar:

- Uygulama macOS tarafinda calisir ve kayit alir
- `Quick Kayit` ve `Tam Kayit` akislari net dosya adlari uretir
- GitHub Release etiketi olusturuldu
- GitHub Pages indirme sayfasi hazirlandi
- Ilk kullanim kilavuzu eklendi
- Windows kontrol listesi eklendi
- Destek / SSS sayfasi eklendi
- Guvenlik politikasi gercek proje metnine cevrildi
- Public issue template'leri eklendi
- Katki rehberi eklendi

## Beklenen Kayit Dosyalari

Quick kayit:

- `quick_take_YYYYMMDD_HHMMSS.mp3`
- `quick_take_YYYYMMDD_HHMMSS_vocal.wav`

Tam kayit:

- `guitar_mix_YYYYMMDD_HHMMSS.mp3`
- `guitar_mix_YYYYMMDD_HHMMSS_vocal.wav`

Not:

- Dosyalar cogu zaman dogrudan Masaustu'ne yazilir
- Yeni klasor olusmamasi normaldir
