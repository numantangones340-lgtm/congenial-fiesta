Bu dosya, uygulamayi GitHub, sosyal medya, mesajlasma uygulamalari veya kisa tanitim alanlarinda duyurmak icin hazir metinler icerir.

## Kisa Repo Aciklamasi

Turkce arayuzlu gitar/vokal kayit uygulamasi. macOS ve Windows icin hizli mikrofon testi, MP3 export ve tek tik quick kayit sunar.

## GitHub Release Duyurusu

`Guitar Amp Recorder v1.1.10` yayinda.

Bu surumle birlikte:

- paketlenmis macOS uygulamasinda mikrofon izin akisi duzeltildi
