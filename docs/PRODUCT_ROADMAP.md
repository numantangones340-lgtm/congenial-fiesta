# Product Roadmap

Bu not, uygulamayi "kolay kurulan, kolay kullanilan ve guvenle indirilen" bir masaustu urune donusturmek icin ana basliklari toplar.

## Hedefler

- Tek tik kurulum ve acilis
- macOS ve Windows icin net indirilebilir paketler
- Basit arayuz, varsayilanlarla hizli kayit
- Guven veren release ve indirme sayfalari
- Ses kalitesini bozmadan kullanimi sade tutmak

## Kisa Vade

1. Release script'lerini eksiksiz hale getir
2. GitHub Releases ve GitHub Pages akisini dogrula
3. README kurulum adimlarini sadelestir
4. Temel smoke/build kontrollerini CI icine sabitle

## Orta Vade

1. Hazir signed/notarized macOS build yayinla
2. Windows build paketini de ayni kalite seviyesine getir
3. Ilk acilista kurulum yardimcisi veya hizli baslangic akisi ekle
4. Kullanici preset ve export akislarini daha da sadeleştir

## Uzun Vade

1. Otomatik guncelleme akisi
2. Resmi urun sayfasi ve sade indirme landing page
3. Kurulum dosyasi / installer deneyimi
4. Daha genis cihaz uyumlulugu ve kalite presetleri
